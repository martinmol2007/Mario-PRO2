## Controles del juego

| Acción    | Tecla |
|----------|------|
| Saltar   | `SPACE` |
| Izquierda| `A` |
| Derecha  | `D` |
| Pausar   | `P` |
| Reiniciar   | `R` |
| Suicidarse   | `K` |
| Curarse   | `H` |
| Restar Vida   | `-` |
| Monedas   | `M` |
| Quitar Moneda   | `N` |
